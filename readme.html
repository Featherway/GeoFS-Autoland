GeoFS Autoland

GeoFS Autoland allows users to land most aircraft automatically in GeoFS Flight Simulator. Many videos have been made to assist the end user, so it is suggested that you watch the tutorials at this channel. The tutorials encompass installing autoland, making an autoland FMC route, and using autoland. There is also a document here that explains how to make an FMC route for autoland.

Bug Reporting and other Github Things

To report a bug, suggest features, or get a route added, go to our github page and open an issue. There are templates for bug reporting, feature adding, and route adding. Please use those for issues.
